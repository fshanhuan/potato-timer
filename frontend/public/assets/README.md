# Assets 目录说明

## 目录结构
public/assets/
├── icons/                    # 图标 ( .png 支持透明底)
│   ├── timer-icon.png        # 计时环中间图标
│   ├── short-break.png       # 短休息图标
│   ├── long-break.png        # 长休息图标
│   └── alert-icon.png        # 紧急提醒图标
├── images/                   # 背景图和头像 .jpg
│   ├── bg1.jpg, bg2.jpg      # 内置背景图
│   └── default-avatar.jpg    # 默认头像
├── audio/                    # 音频文件
│   ├── sound1.mp3, sound2.mp3, sound3.mp3  # 内置提示音
│   └── alert.mp3             # 默认提醒音
└── data/                     # 本地数据存储
    ├── image-paths.json      # 记录用户自定义图片路径
    └── audio-paths.json      # 记录用户自定义音频路径
